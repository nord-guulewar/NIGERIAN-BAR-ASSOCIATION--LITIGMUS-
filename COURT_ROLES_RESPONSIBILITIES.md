# 🏛️ Nigerian Court Roles & Responsibilities - NBA LITIGMUS

## Overview

This document outlines the specific duties and responsibilities of each court officer in the Nigerian judicial system, following the hierarchy and procedures established by Nigerian court rules.

---

## 📋 Court Hierarchy & Case Flow

### Case Filing Process (Nigerian Courts)

```
1. CASHIER/ACCOUNTANT
   ↓ (Receives filing fees)
   
2. CLERK
   ↓ (Receives case documents, verifies completeness)
   
3. REGISTRAR
   ↓ (Generates case number, assigns to judge, checks judge availability)
   
4. SECRETARY
   ↓ (Schedules hearing, notifies lawyers/parties)
   
5. JUDGE
   ↓ (Hears case, delivers judgment)
   
6. RECORDS OFFICER
   ↓ (Archives case files, maintains records)
   
7. BAILIFF
   (Serves summons, enforces court orders)
```

## 👥 Detailed Role Responsibilities

### 1. 📊 **REGISTRAR** (Chief Administrative Officer)

**Primary Duties:**
- Generate case numbers according to state/court format
- Assign cases to judges based on availability and specialization
- Verify judge availability (max 15 cases/day)
- Approve case filings
- Maintain court calendar
- Supervise all court staff
- Ensure compliance with court rules

**Case Number Format:**
```
[COURT]/[STATE]/[LGA]/[YEAR]/[SEQUENCE]

Examples:
- SHC/LA/IKJ/2024/001 (State High Court, Lagos, Ikeja)
- FHC/AB/UMU/2024/045 (Federal High Court, Abia, Umuahia)
- MC/LA/SUR/2024/123 (Magistrate Court, Lagos, Surulere)
```

**Notifications Sent:**
- To Judge: Case assignment
- To Clerk: Case registration confirmation
- To Secretary: Schedule hearing
- To Lawyers: Case number and hearing date

**Dashboard Features:**
- Generate case numbers
- View all judges and their availability
- Assign cases to judges
- View pending case assignments
- Monitor court calendar
- Generate reports

### 3. 📅 **SECRETARY** (Court Secretary)

**Primary Duties:**
- Schedule court hearings
- Notify lawyers of hearing dates
- Send hearing reminders
- Manage judge's calendar
- Coordinate with other courts
- Handle correspondence
- Prepare cause lists

**Notification Schedule:**
- 30 days before hearing: Initial notice
- 7 days before hearing: Reminder
- 1 day before hearing: Final reminder
- After adjournment: New hearing date

**Notifications Sent:**
- To Lawyers (Plaintiff & Defendant): Hearing dates, adjournments
- To Judge: Daily cause list
- To Bailiff: Summons to be served
- To All Parties: Court decisions

**Dashboard Features:**
- Schedule hearings
- Send notifications to lawyers
- View judge's calendar
- Generate cause lists
- Track notification delivery
- Manage adjournments

### 4. 📁 **RECORDS OFFICER** (Records/Registry Officer)

**Primary Duties:**
- Archive case files

- Retrieve case files on request
- Ensure document security
- Digitize records
- Manage file movement

**Record Retention (Nigerian Law):**
- Civil cases: 10 years
- Criminal cases: 15 years
- Land cases: Permanent
- Family cases: 20 years

**Notifications Sent:**
- To Clerk: File retrieved/returned
- To Judge: Case file ready
- To Lawyers: Documents available for collection

**Dashboard Features:**
- Archive case files
- Search and retrieve files
- Track file movement
- Generate file reports
- Manage document requests
- Digital repository

## 📊 Case Number Generation Rules

### Format by Court Type

**Supreme Court (SC):**
```
SC/[YEAR]/[SEQUENCE]
Example: SC/2024/001
```

**Court of Appeal (CA):**
```
CA/[ZONE]/[YEAR]/[SEQUENCE]
Example: CA/L/2024/045 (Lagos Division)
```

**Federal High Court (FHC):**
```
FHC/[STATE]/[YEAR]/[SEQUENCE]
Example: FHC/LA/2024/123
```

**State High Court (SHC):**
```
SHC/[STATE]/[LGA]/[YEAR]/[SEQUENCE]
Example: SHC/LA/IKJ/2024/456
```

**Magistrate Court (MC):**
```
MC/[STATE]/[LGA]/[YEAR]/[SEQUENCE]
Example: MC/LA/SUR/2024/789
```

**Customary Court (CC):**
```
CC/[STATE]/[LGA]/[YEAR]/[SEQUENCE]
Example: CC/LA/EPE/2024/012
```

**Sharia Court (SCA):**
```
SCA/[STATE]/[LGA]/[YEAR]/[SEQUENCE]
Example: SCA/KN/KAN/2024/034
```

### Sequence Rules

- Resets annually (January 1st)
- Unique per court/state/LGA combination
- Zero-padded to 3-4 digits
- Sequential numbering
- No gaps in sequence

## ⚖️ Court Rules Validation

### Filing Requirements

**Civil Cases:**
- Statement of claim
- Affidavit in support
- List of witnesses
- Documentary exhibits
- Proof of service
- Filing fee receipt

**Criminal Cases:**
- Charge sheet
- Proof of evidence
- Witness statements
- Exhibits list
- DPP consent (where required)

### Time Limits (Nigerian Courts)

- **Reply to Statement of Claim:** 30 days
- **Filing Appeal:** 30 days from judgment
- **Service of Processes:** 7 days before hearing
- **Adjournment Notice:** Minimum 7 days
- **Judgment Delivery:** Within 90 days of final address

### Court Sitting Days

- **Regular Courts:** Monday - Friday
- **Vacation Period:** August - September (Limited sittings)
- **Public Holidays:** No sittings
- **Special Sittings:** As directed by Chief Judge
